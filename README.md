# Ed Umbao B2B Lead Tracker

A privacy-first static web tracker for organic B2B lead generation.

## What is different in this version?

This version is self-contained. The CSS and JavaScript are inside `index.html`, so the page will not look like plain text because of a missing stylesheet.

## Safe GitHub Pages upload

Upload only:

- `index.html`
- `README.md`
- `PRIVACY.md`
- `START_HERE.txt`
- `.gitignore`
- `sample_tracker.csv`

Do not upload real CSV files, private exports, spreadsheets, or client lead lists.

## Private workflow

1. Open the public GitHub Pages app.
2. Import your private CSV from your computer.
3. Add, edit, and filter leads.
4. Export the updated CSV.
5. Save the exported CSV outside the GitHub project folder.
6. Clear local data when finished, especially on shared devices.

GitHub is the public tool. Your computer or private drive is the private database.
