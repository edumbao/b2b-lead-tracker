# Privacy Notes

This tracker is designed so the public GitHub repository contains only the app interface, not real lead data.

## What the app does

- Reads imported CSV files in your browser.
- Stores rows in local browser storage when available.
- Exports updated CSV files back to your computer.
- Does not use external scripts, CDNs, analytics, databases, or servers.

## What you should not do

- Do not upload real lead CSVs to GitHub.
- Do not commit exported CSV files.
- Do not store client lead data in a public repository.

## Safer usage

After each work session, export your CSV to a private location and click **Clear local data**.
